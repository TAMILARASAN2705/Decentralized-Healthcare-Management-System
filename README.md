# 🏥 Decentralized Healthcare Management System (DHMS)

A blockchain-based healthcare management system designed to securely store, manage, and verify medical records using decentralized technologies. This project demonstrates how blockchain, cryptographic hashing, role-based access control, and audit logging can improve healthcare data security and transparency.

---

## 📌 Overview

The Decentralized Healthcare Management System (DHMS) provides a secure platform for managing patient records while ensuring data integrity, privacy, and controlled access. The system combines blockchain technology with healthcare workflows to create a tamper-proof medical record management solution.

---

## ✨ Features

### 🔐 Security Features

* SHA-256 hashing for secure record verification
* Blockchain-based immutable ledger
* Secure password hashing and authentication
* Role-based access control
* Comprehensive audit logging

### 👥 User Roles

* **Patient** – Register and manage personal healthcare information
* **Doctor** – Create and access medical records
* **Administrator** – Manage permissions and monitor system activities

### 📋 Medical Record Management

* Create and store patient medical records
* Verify record integrity using blockchain
* Maintain complete medical history
* Secure access authorization

### ⛓️ Blockchain Integration

* Genesis block initialization
* Proof-of-work mining simulation
* Chain validation and integrity checking
* Immutable storage of healthcare transactions

## 🔐 Security Mechanisms

### SHA-256 Hashing

Ensures data integrity and protects sensitive information.

### Blockchain Validation

Every medical record is stored as a blockchain transaction to prevent tampering.

### Audit Logging

Tracks all user activities and access requests.

### Access Control

Only authorized users can view or modify patient records.
